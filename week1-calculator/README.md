# CLI Calculator

A command line interface(CLI) calculator built in Python.

## Features
- Supports +, -, x, /
- Chains calculations (result carries forward)
- Handles division by zero
- Loops until user types "exit"

## How to run
python calculator.py

## What I learned during this
- Functions should return, not print only.
- Using a result variable to chain operations
- Handling edge cases (division by zero)
- Basic Python syntax
