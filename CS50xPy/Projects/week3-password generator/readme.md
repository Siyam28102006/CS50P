### CLI Password Generator

A CLI password generator built in Python.

## Features
- Choose how many passwords to generate
- Choose password length (minimum 6)
- Toggle numbers on/off
- Toggle symbols on/off
- Input validation throughout

## How to run
python password_generator.py

## Example
How many random passwords : 3
Length of password : 10
Do you want numbers in your password [y/n] ? : y
Do you want symbols in your password [y/n] ? : n

1st password is : aBcDeFgHiJ

2nd password is : kLmNoPqRsT

3rd password is : uVwXyZaBcD
