### Student Grade System

A CLI Student Grade generator built in Python.

## Features
- Input multiple subjects and marks
- Calculates total percentage
- Outputs grade (A* to Fail)
- Handles invalid marks (negative or over 100)
- Handles empty input

## Grade Scale
| Grade | Percentage |
|-------|-----------|
| A*    | 90% and above |
| A     | 80% and above |
| A-    | 75% and above |
| B     | 70% and above |
| B-    | 65% and above |
| C     | 60% and above |
| C-    | 55% and above |
| D     | 40% and above |
| Fail  | Below 40% |

## How to run
python grading.py
